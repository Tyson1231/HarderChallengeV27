## Dynamic event update

This build changes stages to six-minute intervals and adds a separate Director system for temporary one-minute world events: Earthquake, Ashfall, Lightning Storm, Meteor Shower, and the rare Blood Moon. Earthquakes never move the camera; they alter nearby terrain in small bursts. Added normal-world fog was not introduced; visibility reduction is limited to Ashfall's temporary darkness pulses.

Backrooms are intentionally not included in this build.
